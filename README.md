# Sabor-do-interior-
Sabor do Interior 062 é uma plataforma digital para um restaurante de culinária caseira em Goiânia. O projeto oferece cardápio dinâmico, pedidos via WhatsApp e informações de entrega. Focado em UX mobile, opera das 11h às 16h. Desenvolvido para ser leve, eficiente e facilitar a conexão entre o tempero do interior e o cliente urbano.
